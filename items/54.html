54 Use grid system for dimensions

Defining everything in dimension for each value is really pain. You can create a grid system for yourself and use it everywhere easily.
Use twitter bootstrap style dimensions. Talk with your designers to adopt this approach as well. With one xml file, you will be able to handle everything easily, instead of creating a different dimen for everything. 
<dimen name='d1'>4</dimen>
<dimen name='d2'>8</dimen>
<dimen name='d3'>12</dimen>
